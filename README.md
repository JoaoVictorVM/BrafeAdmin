# BrafeAdmin

Projeto desenvolvido durante o curso de **CSS Avançado** da Origamid.

A aplicação consiste no **layout de uma área administrativa para uma cafeteria**, simulando um painel interno utilizado para gerenciamento de informações da aplicação.

O diferencial deste projeto é que o mesmo layout foi estilizado utilizando **diferentes abordagens de CSS**, permitindo comparar técnicas de construção de layouts na web.

A versão publicada no **GitHub Pages** foi construída utilizando **Flexbox**, enquanto na pasta `layouts` do repositório é possível encontrar outras versões do mesmo layout desenvolvidas com diferentes técnicas de estilização.

---

### Tech Stack:

- **HTML**
- **CSS (Avançado)**
- **Bootstrap**

---

### Objetivo do Projeto:

- Praticar técnicas avançadas de **CSS**
- Construir o layout de uma **interface administrativa**
- Comparar diferentes abordagens de construção de layout utilizando:
  - **Float**
  - **Flexbox**
  - **Bootstrap**

---

### Aprendizados:

- Construção de layouts utilizando **Flexbox**
- Estruturação de interfaces administrativas
- Comparação entre diferentes técnicas de estilização
- Organização de código utilizando **CSS estruturado**

---
